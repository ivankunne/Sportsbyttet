# Sportsbytte — Brand Reference

## Identity
- Name: Sportsbytte
- Domain: sportsbyttet.no
- Tagline (primary): "Brukt utstyr. Ekte kvalitet. Din klubb."

## Logo rules
- Wordmark: "Sportsbytte" in Fraunces 700
- The "S" is always ~25% larger than other letters
- "S" color: #e8843a (amber) on light and dark backgrounds
- "S" color: #1a3c2e (forest) on amber backgrounds
- Never use a separate icon — the oversized S IS the mark

## Colors
--forest:       #1a3c2e   /* Primary brand, CTAs, headings */
--forest-mid:   #2d5c44   /* Hover states, club badges */
--forest-light: #e8f2ec   /* Backgrounds, tags, tints */
--amber:        #e8843a   /* Accent, logo S, buy CTAs */
--amber-light:  #fdf0e6   /* Event banners, highlights */
--cream:        #f7f5f0   /* Page background */
--ink:          #1a1a1a   /* Body text */
--ink-mid:      #4a4a4a   /* Secondary text */
--ink-light:    #8a8a8a   /* Labels, meta */
--white:        #ffffff   /* Card surfaces, inputs */
--border:       #e2ddd6   /* Card borders, dividers */

## Typography
- Display/headings: Fraunces (Google Fonts) — weights 600, 700
- Body/UI: DM Sans (Google Fonts) — weights 400, 500, 600
- Prices: Fraunces 700, color: --forest
- Never use Inter, Roboto, or system fonts

## Type scale
- Hero:     Fraunces 700, 48–56px
- H1:       Fraunces 700, 28–32px
- H2:       Fraunces 600, 20–24px
- H3:       Fraunces 600, 18px
- Body:     DM Sans 400, 15–16px, line-height 1.6
- Small:    DM Sans 400, 13px
- Label:    DM Sans 600, 11px, uppercase, letter-spacing 0.1em
- Price:    Fraunces 700, 20–32px depending on context

## Buttons
- Primary:  bg #1a3c2e, text white — "Utforsk utstyr", "Registrer din klubb"
- CTA:      bg #e8843a, text white — "Kjøp nå", "Publiser annonse"
- Outline:  border #1a3c2e, text #1a3c2e — secondary actions
- Ghost:    border #e2ddd6, text #4a4a4a — tertiary actions
- Border radius: 8px on all buttons

## Condition badges
- Som ny:     bg #e6f5ee, text #1a6b3e
- Pent brukt: bg #e8f1fb, text #1a4a8a
- Godt brukt: bg #fdf0e6, text #954a00
- Mye brukt:  bg #f2f2f2, text #555555
- Border radius: 20px (pill shape)

## Cards
- Background: #ffffff
- Border: 1px solid #e2ddd6
- Border radius: 12px
- Padding: 16px
- Hover: translateY(-2px), 120ms ease-out

## Spacing (4px base grid)
- xs: 4px | sm: 8px | md: 12px | base: 16px
- lg: 24px | xl: 32px | 2xl: 48px | 3xl: 64px
- Card internal padding: 16px
- Section gaps: 32–48px
- Page margins: 24px mobile, 48px desktop

## Voice & tone
- Language: Norwegian bokmål
- Tone: Direct, warm, community-first — NOT startup hype
- Address users as "du/deg" never "De/Dem"
- Prices written as "4 200 kr" (space separator, lowercase kr)
- Lead with the gear and the club, not the platform
- Good: "Selg ski til folk som faktisk bruker dem"
- Bad: "Vår innovative platform tilbyr en unik opplevelse"

## Motion
- Card hover: translateY(-2px) + shadow, 120ms ease-out
- Page transitions: 200ms fade
- Loading: skeleton shimmer, not spinners
- CTA hover: scale(1.02), 100ms
- No bounce, spring, or confetti

## Photography style
- Natural Norwegian light (overcast, not golden hour)
- Honest condition photos — show wear honestly
- Flat lay on cream/wood for product shots
- No stock photos of smiling models
- No white studio backgrounds
